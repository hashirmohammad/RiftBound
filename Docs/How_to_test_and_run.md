### Running This Project (Godot 4)
Requirements:
- Godot Engine: 4.x (required)
- OS: Windows, macOS, or Linux (64-bit)
- RAM: 4 GB minimum (8 GB recommended)

-This project will not work on Godot 3.x

### Step One Download Godot
- Go to https://godotengine.org/download
- Download Godot 4 for your system
- Extract the file
- Run the executable (no installation needed)

### Step Two Get the Project
- Option A — ZIP
- Click Code → Download ZIP
- Extract the folder

- Option B — Git
- git clone <repository-url>

### Step Three Open the Project
- Launch Godot
- Click Import
- Select project.godot from the project folder
- Click Import & Edit

### Step Four Run
- F5 → Run project
- F6 → Run current scene
- F8 → Stop
- Troubleshooting

- Wrong version error
- Make sure you're using Godot 4.x

- Nothing happens when running
- Go to:
- Project → Project Settings → Application → Run
- Set a Main Scene

- macOS blocked app
- Right-click → Open

-Slow performance
-Try changing:
-Editor → Editor Settings → Rendering → Driver
